Instruções de uso

Bibliotecas necessárias:
    - numpy
    - click

Como executar:
    python simulador_parte3.py --netlist=<path da netlist>

Exemplo:
    python simpulador_parte3.py --netlist=netlist0.txt

Ou executar o programa sem passar argumento para --netlist.

Nesse caso, a netlist será solicitada durante a execução.

Exemplo:
    python simulador_parte3.py
    >> Netlist: netlist0.txt
